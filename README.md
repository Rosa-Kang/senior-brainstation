# 🧠 Senior BrainStation

매일 실시간 뉴스 + 수리/뇌훈련 문제로 두뇌를 단련하는 웹앱

---

## 실행 방법

### 방법 1: 브라우저에서 직접 열기
`index.html` 파일을 더블클릭

### 방법 2: 로컬 서버로 실행 (권장 — API 정상 작동)
```bash
# 이 폴더에서 터미널 열고:
python -m http.server 8080

# 브라우저 주소창에:
http://localhost:8080
```

> **방법 2 권장 이유**: GNews API가 `file://` 프로토콜에서는
> CORS 제한으로 막힐 수 있어요. 로컬 서버를 쓰면 확실히 작동해요.

---

## API 키

| 서비스 | 키 | 용도 |
|--------|-----|------|
| GNews | 앱에 내장됨 | 실시간 뉴스 (무료 100건/일) |
| rss2json | 앱에 내장됨 | 보조 |

키를 바꾸려면: 앱 실행 후 우측 상단 **⚙️ 설정**

---

## 기능

- 📖 **언어영역** — 오늘의 뉴스 4건 (국내/세계/경제/IT) + 시사 질문
- 🔢 **수리영역** — 150개 큐레이션 IQ 문제 + 알고리즘 자동 생성
- 🧠 **뇌 훈련** — 기억력(숫자)/관찰력(다른 글자 찾기) 게임
- ✉️ **이메일 정리** — 오늘의 오답을 이메일로 발송

## 데이터 저장

모든 학습 기록은 **브라우저 localStorage**에 저장돼요.
(서버 없음, 개인정보 외부 전송 없음)
